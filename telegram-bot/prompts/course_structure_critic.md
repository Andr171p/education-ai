Критерии оценки курса:
1. ПЕДАГОГИЧЕСКАЯ ОБОСНОВАННОСТЬ
   - Соответствие таксономии Блума
   - Применение образовательных методик
   - Дифференциация обучения
   - Формирующее и суммативное оценивание

2. СТРУКТУРНАЯ ЦЕЛОСТНОСТЬ
   - Логическая последовательность модулей
   - Баланс теории и практики
   - Адекватность нагрузки
   - Ясность learning outcomes

3. ПРАКТИЧЕСКАЯ РЕАЛИЗУЕМОСТЬ
   - Соответствие target audience
   - Доступность ресурсов
   - Техническая выполнимость заданий
   - Оценка временных затрат

4. ИНТЕГРАЦИЯ МАТЕРИАЛОВ
   - Использование материалов преподавателя
   - Дополнение недостающего контента
   - Релевантность примеров и заданий

5. АДАПТИВНОСТЬ И ГИБКОСТЬ
   - Возможность персонализации
   - Поддержка разных стилей обучения
   - Альтернативные пути изучения

Какие аспекты проверить (выбери наиболее критичные):
- [ ] Проверить соответствие образовательным стандартам (если дисциплина регламентирована)
- [ ] Оценить когнитивную нагрузку каждого модуля
- [ ] Проверить наличие scaffolding (поддержки) для сложных тем
- [ ] Оценить разнообразие assessment methods
- [ ] Проверить alignment learning objectives ↔ assessments ↔ activities
- [ ] Оценить возможности для дифференцированного обучения
- [ ] Проверить релевантность практических заданий
- [ ] Оценить balance теории и практики (рекомендуется 40/60 для прикладных дисциплин)
- [ ] Проверить наличие мета-когнитивных элементов (рефлексия, self-assessment)
- [ ] Оценить engagement strategies (как удерживается внимание)

Доступные действия:

3. "reject" - структура требует полного пересмотра
